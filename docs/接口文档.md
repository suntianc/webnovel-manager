# 网文素材管理系统 - REST API 文档

**版本:** 1.0.0
**更新日期:** 2026-04-25
**基础 URL:** `http://localhost:3000`
**OpenAPI 文档:** `http://localhost:3000/docs`

---

## 概述

本系统为网文素材管理后端服务，提供素材的 CRUD、全文搜索、标签管理等功能。

### 技术栈

- **框架:** FastAPI (Python 3.10+)
- **数据库:** SQLite + FTS5 全文检索
- **数据验证:** Pydantic V2

### 认证说明

本系统为**单用户本地部署**，无认证机制。

### 请求格式

- Content-Type: `application/json`
- 请求体使用 JSON 格式

### 响应格式

直接返回 JSON 数据，无统一包装。

### 错误响应

```json
{
  "detail": "错误描述"
}
```

## 素材接口

### GET /api/materials

获取素材列表，支持分页和筛选。

**查询参数:**

| 参数 | 类型 | 说明 |
|------|------|------|
| category | string | 一级分类 |
| subcategory | string | 二级分类 |
| status | string | 整理状态 |
| keyword | string | 关键词搜索 |
| page | integer | 页码（默认 1） |
| limit | integer | 每页数量（默认 20，最大 100） |

**响应 (200):** `list[MaterialResponse]`

```json
[
  {
    "id": 1,
    "title": "帝国政治体系设定",
    "content": "正文内容...",
    "summary": "摘要",
    "category": "世界观",
    "subcategory": "政治体系",
    "source_type": "手动",
    "source_url": null,
    "status": "已整理",
    "value_score": 5,
    "tags": ["帝国", "权谋"],
    "created_at": "2026-04-25T10:00:00",
    "updated_at": "2026-04-25T10:00:00"
  }
]
```

**MaterialResponse 字段说明:**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 素材 ID |
| title | string | 标题 |
| content | string | 正文内容 |
| summary | string/null | 摘要 |
| category | string/null | 一级分类 |
| subcategory | string/null | 二级分类 |
| source_type | string | 来源类型（默认"手动"） |
| source_url | string/null | 来源链接 |
| status | string | 整理状态（待整理/已整理/采集中/已使用/已归档/已废弃） |
| value_score | int | 价值评分 0-5 |
| tags | list[string] | 标签列表 |
| created_at | string | 创建时间 |
| updated_at | string | 更新时间 |

---

### POST /api/materials

创建新素材。

**请求体:**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| title | string | 是 | 标题 |
| content | string | 是 | 正文内容 |
| summary | string | 否 | 摘要 |
| category | string | 否 | 一级分类 |
| subcategory | string | 否 | 二级分类 |
| source_type | string | 否 | 来源类型（默认"手动"） |
| source_url | string | 否 | 来源链接 |
| status | string | 否 | 整理状态（默认"待整理"） |
| value_score | int | 否 | 价值评分 0-5 |
| tags | list[string] | 否 | 标签列表 |

**请求示例:**

```json
{
  "title": "帝国政治体系设定",
  "content": "正文内容...",
  "summary": "关于帝国政治体系的设定",
  "category": "世界观",
  "subcategory": "政治体系",
  "status": "待整理",
  "value_score": 5,
  "tags": ["帝国", "权谋"]
}
```

**响应 (201):** `MaterialResponse`

```json
{
  "id": 1,
  "title": "帝国政治体系设定",
  "content": "正文内容...",
  "summary": "关于帝国政治体系的设定",
  "category": "世界观",
  "subcategory": "政治体系",
  "source_type": "手动",
  "source_url": null,
  "status": "待整理",
  "value_score": 5,
  "tags": ["帝国", "权谋"],
  "created_at": "2026-04-25T10:00:00",
  "updated_at": "2026-04-25T10:00:00"
}
```

### GET /api/materials/{material_id}

获取素材详情。

**路径参数:**

| 参数 | 类型 | 说明 |
|------|------|------|
| material_id | int | 素材 ID（必须 > 0） |

**响应 (200):** `MaterialResponse`

**响应 (404):**

```json
{
  "detail": "Material not found"
}
```

---

### PUT /api/materials/{material_id}

更新素材。所有字段可选，只更新提供的字段。

**路径参数:**

| 参数 | 类型 | 说明 |
|------|------|------|
| material_id | int | 素材 ID |

**请求体:** 同 POST /api/materials，所有字段可选

**请求示例:**

```json
{
  "status": "已整理",
  "value_score": 4
}
```

**响应 (200):** `MaterialResponse`

**响应 (404):**

```json
{
  "detail": "Material not found"
}
```

---

删除素材。

| material_id | int | 素材 ID |

```json

}
```

## 搜索接口

### GET /api/search

FTS5 全文搜索。

**查询参数:**

| 参数 | 类型 | 说明 |
|------|------|------|
| q | string | 搜索关键词（最少 1 字符） |
| category | string | 一级分类 |
| subcategory | string | 二级分类 |
| status | string | 整理状态 |
| tags | string | 标签筛选（逗号分隔） |
| limit | int | 返回数量（默认 20，最大 100） |

**响应 (200):** `list[SearchResult]`

```json
[
  {
    "id": 1,
    "title": "帝国政治体系设定",
    "snippet": "正文内容片段...",
    "rank": 1.5,
    "category": "世界观",
    "subcategory": "政治体系",
    "tags": ["帝国", "权谋"],
    "status": "已整理",
    "created_at": "2026-04-25T10:00:00"
  }
]
```

**SearchResult 字段说明:**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 素材 ID |
| title | string | 标题 |
| snippet | string | 内容片段（高亮） |
| rank | float | 相关性评分 |
| category | string/null | 一级分类 |
| subcategory | string/null | 二级分类 |
| tags | list[string] | 标签列表 |
| status | string/null | 整理状态 |
| created_at | string/null | 创建时间 |

## 标签接口

### GET /api/tags

获取所有标签。

**响应 (200):** `list[TagResponse]`

```json
[
  {
    "id": 1,
    "name": "帝国",
    "tag_type": "内容标签",
    "created_at": "2026-04-25T10:00:00"
  },
  {
    "id": 2,
    "name": "权谋",
    "tag_type": "内容标签",
    "created_at": "2026-04-25T10:00:00"
  }
]
```

**TagResponse 字段说明:**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 标签 ID |
| name | string | 标签名称 |
| tag_type | string | 标签类型（默认"内容标签"） |
| created_at | string | 创建时间 |

---

### POST /api/tags

创建标签。

**请求体:**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| name | string | 是 | 标签名称（1-50 字符） |
| tag_type | string | 否 | 标签类型（默认"内容标签"） |

**请求示例:**

```json
{
  "name": "新标签",
  "tag_type": "内容标签"
}
```

**响应 (201):** `TagResponse`

```json
{
  "id": 3,
  "name": "新标签",
  "tag_type": "内容标签",
  "created_at": "2026-04-25T12:00:00"
}
```

**响应 (400):**

```json
{
  "detail": "Tag already exists"
}
```

---

**路径参数:**

| tag_id | int | 标签 ID |


## 统计接口

### GET /api/stats

获取素材统计数据。

**响应 (200):**

```json
{
  "total": 100,
  "total_tags": 50,
  "by_status": {
    "待整理": 20,
    "已整理": 50,
    "采集中": 10,
    "已使用": 15,
    "已归档": 3,
    "已废弃": 2
  },
  "by_category": {
    "世界观": 30,
    "人物": 25,
    "剧情": 20,
    "场景": 15,
    "文风": 5,
    "金手指": 3,
    "资料考据": 2
  },
  "avg_score": 4.2,
  "recent_count": 5
}
```

**StatsResponse 字段说明:**

| 字段 | 类型 | 说明 |
|------|------|------|
| total | int | 素材总数 |
| total_tags | int | 标签总数 |
| by_status | dict | 按状态统计 |
| by_category | dict | 按分类统计 |
| avg_score | float | 平均价值评分 |
| recent_count | int | 近 7 天新增素材数 |

## HTTP 状态码

| 状态码 | 说明 |
|--------|------|
| 200 OK | 请求成功 |
| 201 Created | 资源创建成功 |
| 204 No Content | 资源删除成功（无返回内容） |
| 400 Bad Request | 请求参数错误 |
| 404 Not Found | 资源不存在 |
| 422 Unprocessable Entity | 数据验证失败 |
| 500 Internal Server Error | 服务器内部错误 |
